jQuery.simpleTypeahead

Adds a typeahead dropdown menu to any text input. View Github repository

<input class="typeahead" />

<script>
  $(".typeahead").simpleTypeahead({
    items: ["Javascript", "Java", "CSS", "HTML"]
  });
</script>

Options

maxNbMatches

Set the number of matches to show in the dropdown.

$(".typeahead").simpleTypeahead({
  items: ["Javascript", "Java", "CSS", "HTML"],
  maxNbMatches: 1
});

mustMatch

Set the maximum number of tags that can be entered in the input field.

$(".typeahead").simpleTypeahead({
  items: ["Javascript", "Java", "CSS", "HTML"],
  maxNbMatches: 1,
  mustMatch: false,
  onNoMatchesItem: function(value) {
    return $('<li class="typeahead-result">' + text + ' (Add Result)</li>');
  }
}
});

Methods

getTags

Returns an array of the entered tag values.

$(".tagger").simpleTagger("getTags");
